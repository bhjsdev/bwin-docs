# `addPane`

## Description

Add a pane to an existing pane.

```js
binaryWindow.addPane(targetPaneId, options);
```

## Parameters

### Function parameters

| Name           | Type   | Description                                          |
| -------------- | ------ | ---------------------------------------------------- |
| `targetPaneId` | string | Target pane's ID that the new pane is added to.      |
| `options`      | object | Options for new pane. e.g., `size`, `position`, etc. |

### `options` details

| Name       | Type                                | Description                                                          |
| ---------- | ----------------------------------- | -------------------------------------------------------------------- |
| `content`  | string \| DOM Node \| ReactNode[^1] | Content of the new pane                                              |
| `id`       | string                              | ID of the new pane                                                   |
| `position` | string                              | Position of the new pane. One of `top`, `right`, `bottom`, or `left` |
| `size`     | number \| string                    | Size of the new pane. e.g., 100, 0.5, "50%"                          |
| `title`    | string \| DOM Node \| ReactNode[^1] | Title of the new pane                                                |

[^1]: `DOM Node` is for vanilla Javascript, and `ReactNode` is for React.

## Returns

**Type**: `Sash` object

**Description**: Newly created Sash object.

## Code snippets

### Javascript

```html
<div id="container"></div>
```

```js
const binaryWindow = new BinaryWindow();

binaryWindow.mount(document.getElementById('container'));

binaryWindow.addPane('target-pane-id', {
  position: 'bottom',
  size: 120,
  content: Object.assign(document.createElement('div'), { innerHTML: 'Hello World' }),
});
```

### React

```jsx
import { useEffect, useRef } from 'react';
import { Window } from 'bwin-react';

function App() {
  const windowRef = useRef(null);

  useEffect(() => {
    windowRef.current.addPane('target-pane-id', {
      position: 'bottom',
      size: 120,
      content: <div>Hello World</div>,
    });
  }, []);

  return <Window ref={windowRef} />;
}
```
