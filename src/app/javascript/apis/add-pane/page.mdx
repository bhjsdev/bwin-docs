# Add a pane

## `addPane`

Add a pane to an existing pane.

```js
binaryWindow.addPane(paneId, options);
```

### Parameters

#### Function parameters

| Name      | Type   | Description                                          |
| --------- | ------ | ---------------------------------------------------- |
| `paneId`  | string | Target pane's ID that the new pane is added to.      |
| `options` | object | Options for new pane. e.g., `size`, `position`, etc. |

#### `options` details

| Name       | Type             | Description                                                          |
| ---------- | ---------------- | -------------------------------------------------------------------- |
| `content`  | string\|DOM Node | Content of the new pane                                              |
| `position` | string           | Position of the new pane. One of `top`, `right`, `bottom`, or `left` |
| `size`     | number\|string   | Size of the new pane. e.g., 100, 0.5, "50%"                          |

### Returns

**Type**: `Sash` object

**Description**: Newly created Sash object.

### Example

<br />
<IFrame path="add-pane" />

```html
<button id="add-pane">Add pane</button>
<div id="container" style="width: 400px; height: 260px"></div>
```

```js
const bwin = new BinaryWindow({
  fitContainer: true,
  children: [
    { position: 'left', id: 'pane-1', content: 'Pane 1' },
    {
      position: 'right',
      id: 'pane-2',
      content: 'Pane 2',
    },
  ],
});

bwin.mount(document.getElementById('container'));

document.getElementById('add-pane').addEventListener('click', () => {
  bwin.addPane('pane-2', {
    content: Object.assign(document.createElement('em'), { innerHTML: 'Pane 3' }),
    position: 'bottom',
    size: 120,
  });
});
```
